# Music assets for reels

Drop royalty-free MP3/M4A/WAV files in this folder. At render time, the reel
pipeline picks one at random, fades it in over 0.8 s, and fades it out at the
end of the 25 s reel.

If this folder is **empty**, reels render without audio. Instagram's Reels
composer lets viewers add a soundtrack in-app, so empty is a valid fallback —
but your own cinematic track is obviously better.

## Recommended sources (all free, no attribution needed)

- <https://pixabay.com/music/> — filter by "Cinematic" / "Ambient" / "Trap"
- <https://freemusicarchive.org/> — filter by CC0 / Royalty Free
- <https://incompetech.com/music/royalty-free/music.html> (attribution required; include in caption)

## Suggested picks for the FinanceBlueprint vibe

Look for tracks tagged "cinematic", "corporate", "ambient piano", "dark trap",
"luxury". Duration doesn't matter — we trim to 25 s. BPM around 70-110 works
best with the 5-second scene cadence.

## Naming

Any filename is fine. Lowercase and underscores preferred, e.g.:

    cinematic_rise.mp3
    ambient_focus.mp3
    dark_luxury.mp3

## Licence reminder

Before committing a track, confirm the licence explicitly says it's safe to
use for commercial social-media posts without attribution (or with attribution
that you include in the caption).
